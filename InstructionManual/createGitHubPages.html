How to create github page

  1. Open github.com
  2. create a new account
  3. create new repository
  4. In the code page click on 'add existing file' and select your index.html file from your system (Note: create index.html file ahead of time in your system), add it and commit it
  5. edit the index.html in VS Code or any editor
    1. add your site name and your image in index.html using html code
    2. save file
  6. go to github and choose add file -> upload files menu
  7. select the updated file and add it and commit it
  8. navigate to the site link and view it
  9. the site may not open up, hence use the instructions provided in this page to publish your site.
    How to publish your site
  10. Note it may take few minutes for the changes to be effective and visible in the site
  11. Marvel at the new site you created all by yourself after doing this course. Seriously! Pat on your back and Smile, You Did It!!!
PS: Couple of other tips.
  • Note: if you want to create a subfolder, click on create new file, and in the file name, include name of the subfolder followed by file name. E.g.: subfolder/newfile.html
